
#  To-Do List Application (Python)

A simple yet powerful command-line To-Do List application built with Python.
This project allows users to efficiently manage daily tasks with an interactive menu system.

---

##  Features

*  Add new tasks
*  View all tasks in a structured list
*  Remove tasks by selecting task number
*  Continuous interactive menu system
*  Input validation for better reliability

---

##  Built With

* **Python 3**
* Core Concepts:

  * Lists
  * Loops (`while`)
  * Conditional Statements
  * Exception Handling (`try/except`)

##  How to Run

### 1. Clone the repository

```bash
git clone https://github.com/your-username/todo-list.git
```

### 2. Navigate to project folder

```bash
cd todo-list
```

### 3. Run the program

```bash
python todo.py
```

---

##  Usage

```
~~~TO-DO LIST~~~

>>>Select Your Choice<<<
01. Add Task
02. View Tasks
03. Remove Task
04. Exit
```

Follow the menu to manage your tasks easily.
